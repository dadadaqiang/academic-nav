# 🌐 全球学术科研导航

> 发现国外大学生 · 教授 · 研究者的网络聚集地

收录 **20+ 个平台**，涵盖 5 大分类，帮助你快速融入国际科研圈子。

## 📂 分类

| 分类 | 说明 |
|------|------|
| 🔬 **专业学术社交网络** | ResearchGate、Academia.edu 等学术社交平台 |
| 🎓 **学生×教授连接平台** | 验证制师生对接、合作项目平台 |
| 💬 **综合学术社区/论坛** | 中文及国际学术论坛 |
| 🐦 **社交媒体** | Twitter/X、Reddit、LinkedIn 等科研活跃阵地 |
| 📹 **视频/即时通讯/其他** | YouTube 学术频道、Slack/Discord 社群等 |

## ✨ 特性

- **分类筛选** — 按类别快速过滤平台
- **★ 推荐标记** — 重点平台一目了然
- **标签系统** — 每个平台标注核心特点和适用场景
- **使用建议** — 附四步实操指南，从零开始融入国际科研圈

## 🚀 使用

直接打开 `index.html` 即可浏览，无需任何构建工具。

```bash
# 本地预览
open index.html
```

也可部署到任意静态托管服务（GitHub Pages、Cloudflare Pages、Netlify 等）。

## 🛠 技术栈

- 纯 HTML + CSS + JavaScript
- 无外部依赖
- 响应式设计，适配手机和桌面
